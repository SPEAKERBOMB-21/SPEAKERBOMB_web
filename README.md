SPEAKERBOMB UI: Project Overview
SPEAKERBOMB is a Python-based Streamlit e-commerce dashboard. It features a dynamic product filtering engine, real-time cart calculations, and a simulated payment gateway. Designed for audiophiles, it offers a sleek interface to browse high-end audio gear from Portable to Concert series.

Quick Start:

PLEASE ENSURE YOU HAVE INSTALLED STREAMLIT (or else it won't work...)

Install Streamlit: pip install streamlit

Run: streamlit run SPEAKERBOMB_UI_design.py
